## 未実装項目サマリ

### Image Gen Console: Wildcard / Dynamic Prompts

プロンプト内に変数記法を埋め込み、投入時にクライアント側で展開する機能。

**仕様（ドラフト・後日詰める）**
- インライン記法: `{red|blue|green}` → ランダム選択
- 重み付き: `{2::red|1::blue}` → 2:1 で red 優先
- 連番: `{1-3}` → 1〜3 の整数ランダム
- ファイル参照: `__hair__` → `wildcards/hair.txt` から 1 行ランダム選択
- バッチ生成と組み合わせると "枚ごとに違う組み合わせ" を試せる

**設計検討事項**
- ファイル参照の置き場所（DB? 静的 JSON? `wildcards/` フォルダ?）
- 展開はクライアント / サーバどちら? → 再現性のため展開後の文字列を `params.WILDCARD_TRACE` に残すと良い
- 入れ子サポート（`{a|{b|c}}`）の必要性
- バッチ時の seed と整合させるか（同じ seed なら同じ展開にすべきか）
